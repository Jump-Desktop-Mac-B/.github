## Overview

Jump Desktop Advanced is a fast and secure remote desktop application that connects to Windows, macOS, and Linux computers from anywhere using RDP, VNC, and the proprietary high-performance Fluid protocol, providing responsive remote access that feels close to sitting in front of the remote machine. It handles the full range of remote access scenarios from occasional personal use through enterprise-scale deployment with consistent performance and security. The application automatically selects the optimal protocol and connection parameters based on network conditions, ensuring the best possible experience without manual configuration.

The Fluid protocol delivers high-frame-rate remote desktop performance with adaptive quality that responds to network conditions in real time, providing smooth scrolling, responsive cursor movement, and video playback that RDP and VNC struggle to match over variable connections. Fluid uses modern compression and rendering techniques designed for today's high-resolution displays and bandwidth capabilities. The protocol handles Retina-resolution displays natively, providing sharp text and UI elements at any display scaling.

Multi-protocol support connects to Windows machines via RDP with full feature support including RemoteApp, multi-monitor spanning, and audio redirection, while VNC connects to any platform running a VNC server. Protocol-specific settings are configured per connection, and Jump Desktop handles authentication, encryption, and session management for each protocol transparently. The application stores connection credentials securely in the system keychain.

Team management through Jump Desktop Connect enables centralized computer registration where remote machines are accessible by team members through a managed directory without requiring VPN configuration, firewall port opening, or IP address knowledge. The relay infrastructure handles NAT traversal and connection brokering automatically, making remote access work across any network topology. Two-factor authentication and role-based access controls ensure that remote access is both convenient and secure.

For IT professionals, remote workers, and anyone who needs reliable access to computers that are not physically present, Jump Desktop Advanced is the remote access tool that combines protocol breadth, performance optimization, and enterprise security in a client that makes remote computing feel local. Its ability to handle any remote access scenario with consistent quality makes it the single remote desktop client that replaces multiple specialized tools.

## Performance

Jump Desktop balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
